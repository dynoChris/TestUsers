# TestUsers

Fetch random users from Random User API.

### Features
  - Fetching data from server.
  - Passing data to another activity.
  - Support landscape mode.
  - Saving state after rotate. 
  
### Technologies
  - MVP (Moxy)
  - RxJava 2
  - Dagger 2
  - Retrofit
  - Picasso
